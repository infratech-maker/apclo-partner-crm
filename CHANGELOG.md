## [1.0.0] - 2024-12-24

### Added
- Initial GitHub integration setup
- Comprehensive .gitignore for .NET/Avalonia, Next.js/TypeScript, and Python
- .gitattributes for line ending normalization
- GitHub CI/CD workflows
- Issue templates and PR template
- Dependabot configuration
- Project root README.md
- UberEats scraper scripts and documentation
  - Background scraper scripts
  - UberEats bulk collection guide
  - Phase 2 README documentation
  - Automated UberEats list collection script

### Infrastructure
- Docker Compose setup (PostgreSQL, Redis)
- Multi-tenant support with Row Level Security (RLS)
- Next.js 15 CRM platform
- Avalonia .NET 8.0 desktop application
